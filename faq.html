FAQ

Da Basics
What is Re-Dream?

Re-Dream is a YSWS that allows you to get funds to spend on your project so that you can actually but your game into production!

Who can participate?

Anybody ages 13-18 can participate worldwide, regardless of skill!

And all of this is free?

Yep!! Hack Club is a 501(c)(3) non-profit organization that’s trying to get more teens into coding! They’ve run a bunch of programs in the past, such as a 180-person hardware hackathon at GitHub HQ!

Projects
What if I’m new to coding?

That’s perfectly fine! You don’t have to have any experience to participate, this could end up as a really good experience to learn how to code!

What can I make?

You can make any game that can be played on Itch.io or Steam.

Can I make more than one project?

Of course! Though for the purpose of this YSWS it’s probably good to not split between too many projects, you totally can!

Can I make more than one project?

Yes, you can submit any previous project you’ve start before but only time spent after [mm/dd/yyyy] will count towards you hours!.

Does my code have to be open source?

Yes, your project does have to be open source on github or some similar software.

Other
Can I use AI?

Using AI to make music or art is not allowed, but for code you can use up to 30% AI. Although, your project cannot be “AI slop” (overly vibecoded with seemingly very little human input)

Can I submit to this event and another Hack Club program?

Sorry, nope! "Double-dipping" is not allowed. If you make a project for this event, you can only submit it here unless another program specifically states otherwise.

Can I get involved or contribute?

Yes, I love this question! If you want to help with organizing, review, ect, you can reach out to me on slack {@Skylar} and I can see if we have anything you can help out with!

Still have questions?

Reach out at email@hackclub.com or #re-dream-help we'd love to hear from you!
